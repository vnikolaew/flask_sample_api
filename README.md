# Steps for running the API locally
 1. Make sure you have Python 3.10, Docker, and docker-compose installed locally on your machine. You can download and install Python from [python.org](https://www.python.org) and Docker from [docker.com](https://www.docker.com).
 2. Clone this repository to your local machine:
 ```
 git clone https://github.com/your_username/flask_sample_api.git
 ```
 3. Navigate to the project's root directory:
 ```
 cd flask_sample_api
 ```
 4. Run the following command to spin up the Flask API and Postgres containers using docker-compose:
 ```
 docker compose up .
 ```
